# Bikeshare Project

With the bikeshare project the user is able to get statistics for the usage of bikeshares for the 3 different cities new york city, chicago and washington dc.
The user is able to choose the city to be analysed and is also able to filter by month and/or day if wished. The program then give the user statistics as output

## Information about how to use your project

As user:

- Run the python bikeshare.py in the terminal
- Choose a city
- Choose a filter (month,day, both, none) 
- (optional) Select month, or day
