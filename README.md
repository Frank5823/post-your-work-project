# Bikeshare Project

With the bikeshare project the user is able to get statistics for the usage of bikeshares for the 3 different us cities: New York City, Chicago and Washington dc.
The user is able to choose the city to be analysed and is also able to filter by month and/or day if wished. The program then give the user statistics as output. It calculates statistics on the most frequent times of travel, the most popular stations and trips, the total and average trip duration as well as on bikeshare users.

You will be asked to select if you want to filter by day, month, both, or not at all:  
- Choose a filter (month,day, both, none)
